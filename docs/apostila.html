Introdução a HTML e CSS

  • HTML é usado para estruturar o conteúdo de uma página, como títulos, parágrafos e imagens.
  • CSS é responsável pela aparência da página, cores, fontes, layout e etc.

Tags HTML

As tags no HTML são elementos que você insere em um documento HTML para marcar e estruturar o conteúdo. Elas são usadas para definir como o navegador da web deve interpretar e exibir diferentes partes do seu documento.

  • Tag de Abertura: É usada para iniciar um elemento e define seu tipo. Por exemplo, < p > é uma tag de abertura que indica o início de um parágrafo.
  • Tag de Fechamento: É usada para encerrar um elemento e geralmente é igual à tag de abertura, mas inclui uma barra antes do nome do elemento. Por exemplo, < / p> é a tag de fechamento que indica o fim de um parágrafo.
  • < html >: A tag raiz que envolve todo o conteúdo da página.
  • < head >: Contém informações sobre a página, como o título exibido na guia do navegador.
  • < title >: Define o título da página que aparece na guia do navegador.
  • < body >: Contém o conteúdo visível da página, como texto, imagens e links.
  • < h1 >, < h2 >, < h3 >, ...: São cabeçalhos usados para títulos e subtítulos
  • < p >: Define um parágrafo de texto.
  • < a >: Cria um link para outra página ou recurso na web.
  • < img >: Exibe uma imagem na página.
  • < ul > e < ol >: Criam listas não ordenadas (com marcadores) e listas ordenadas (numeradas).
  • < li >: Define cada item da lista (< ul > ou < ol >).
  • < div >: É uma caixa de conteúdo usada para agrupar elementos e aplicar estilos.

CSS

O CSS é a linguagem usada para estilizar páginas da web e controlar sua aparência.

  • Os seletores são usados para selecionar elementos HTML aos quais você deseja aplicar estilos. Por exemplo, você pode usar h1 como um seletor para selecionar todos os cabeçalhos < h1 > em sua página.
  • As propriedades são características que você deseja estilizar, como cor, tamanho de fonte, margem, preenchimento e muito mais.
  • Os valores são atribuídos às propriedades e definem como o estilo será aplicado. Por exemplo, color: blue define a cor do texto como azul.
  • Você pode controlar a posição de elementos na página usando propriedades como position, top, bottom, left e right.
  • hover para estilizar um link quando o cursor passa por cima.
  • E muito mais!!!
